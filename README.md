# Multi-Agent AI System: Autonomous Decision Engine

An advanced AI system where multiple intelligent agents collaborate to perform research, analysis, and decision-making using Large Language Models (LLMs) and Retrieval-Augmented Generation (RAG).

This project simulates a real-world AI workflow where different agents act like a team to solve complex problems.

---

## 🚀 Key Features

- Multi-agent architecture (Planner, Research, Analysis, Decision)
- Task decomposition and delegation
- Knowledge retrieval using RAG
- Context-aware decision-making
- Memory system for storing past interactions
- Modular and scalable design

---

## 🧠 Architecture

User Query → Planner Agent → Research Agent → Analysis Agent → Decision Agent → Memory Storage

---

## 🛠️ Technologies Used

- Python  
- Transformers (LLMs)  
- SentenceTransformers (Embeddings)  
- FAISS (Vector Database)  
- Machine Learning & NLP  
- JSON (Knowledge Base)
